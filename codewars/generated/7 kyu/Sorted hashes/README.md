## Sorted hashes
[Sorted hashes](https://www.codewars.com/kata/sorted-hashes)
Category : reference

### Description
Complete the solution so that it sorts an array of hashes passed in by the hash key specified. 

Example: 

```ruby
solution([{'a' => 2}, {'a' => 1}], 'a')
# => [{'a' => 1}, {'a' => 2}]]

solution([{a: 1}, {a: 5}, {a: 3}], :a)
# => [{a:1}, {a: 3}, {a: 5}]

```
