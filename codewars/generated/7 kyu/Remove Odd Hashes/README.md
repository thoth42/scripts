## Remove Odd Hashes
[Remove Odd Hashes](https://www.codewars.com/kata/remove-odd-hashes)
Category : algorithms

### Description
To complete this kata remove any hash whose two keys sum to an odd number.


example: 
```ruby
remove_odd_hashes(
  [ {a: 5, b: 5}, 
    {a: 3, b: 4}, 
    {a: 2, b: 0}, 
    {a: 2, b: 1}], 
  :a, :b)
  
# returns [{a: 5, b: 5}, {a: 2, b: 0}]
```
