## Pyramid Array
[Pyramid Array](https://www.codewars.com/kata/pyramid-array)
Category : algorithms

### Description
Write a function that when given a number >= 0, returns an Array of ascending length subarrays.


pyramid(0)
=> [ ]

pyramid(1)
=> [  [1]  ]

pyramid(2)
=> [  [1],  [1, 1]  ]

pyramid(3)
=> [  [1],  [1, 1],  [1, 1, 1]  ]
