## Square(n) Sum
[Square(n) Sum](https://www.codewars.com/kata/square-n-sum)
Category : reference

### Description
Complete the squareSum method so that it squares each number passed into it and then sums the results together. 

For example:
```javascript
squareSum([1,2, 2]); // should return 9
```
```coffeescript
squareSum([1,2, 2]) # should return 9
```
```ruby
squareSum([1,2, 2]) # should return 9
```
